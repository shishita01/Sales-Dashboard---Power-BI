# Project Overview
This project focuses on building an interactive sales performance dashboard using Power BI.
The dashboard helps analyze sales, profit, and profitability trends across different categories, regions, segments, and time periods.

The goal of this project is to demonstrate data analysis, visualization, and business insight generation skills using a real-world retail dataset.



# Objectives
Analyze total sales and total profit
Track year-wise sales and profit trends
Compare sales and profit ratio across regions
Identify top-performing categories and sub-categories
Enable interactive filtering by category, region, and date



# Tools & Technologies
Power BI Desktop
CSV Dataset (Superstore Sales)
(Optional) Python (Pandas) for preprocessing

These filters allow users to dynamically explore the data and focus on specific business questions.



# Key Insights
Technology category contributes the highest sales and profit, making it the most profitable segment.
West region leads in total sales, while Central region shows a lower profit ratio.
Sales and profit have increased steadily over the years, indicating business growth.
First Class shipping mode has the highest profit ratio, suggesting better margin efficiency.
